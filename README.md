# FizzBuzzKata with Erlang

`runner.sh` to run all tests.

# Execute
```
      
  $ cd src/
  $ erl
  1> c(fizzbuzz).
  {ok,fizzbuzz}
  2> c(collector).
  {ok,collector}
  3> c(converter).
  {ok,converter}
  4> fizzbuzz:start(15).
  ["1","2","Fizz","4","Buzz","Fizz","7","8","Fizz","Buzz",
   "11","Fizz","13","14","FizzBuzz"]
  5> 

```
